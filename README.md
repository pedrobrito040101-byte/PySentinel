# IT-Support-Security-Automation

## 🚀 Visão Geral do Projeto

Este repositório serve como um portfólio prático e uma base de conhecimento para demonstrar habilidades em **Suporte Técnico Nível 2 (N2)** e **Cibersegurança Júnior**. O objetivo é apresentar soluções para problemas comuns de TI, scripts de automação para otimização de tarefas e guias de hardening para fortalecer a segurança de sistemas.

Desenvolvido por Pedro Henrique de Brito, este projeto reflete a transição e aprofundamento em cibersegurança, mantendo a expertise em suporte técnico.

## ✨ Conteúdo do Repositório

O projeto está organizado nas seguintes seções:

### 📜 Scripts de Automação

Coleção de scripts (PowerShell, Bash) para automatizar tarefas rotineiras de suporte e segurança, como:

*   Limpeza de logs e arquivos temporários.
*   Verificação de serviços e processos críticos.
*   Auditoria básica de configurações de segurança.
*   Gerenciamento de usuários e permissões.

### 📝 Guias de Troubleshooting N2

Documentação detalhada com procedimentos passo a passo para diagnosticar e resolver problemas complexos, incluindo:

*   Resolução de problemas de conectividade de rede.
*   Diagnóstico de falhas em sistemas operacionais (Windows/Linux).
*   Troubleshooting de banco de dados SQL (consultas e otimizações).
*   Recuperação de sistemas após incidentes.

### 🔒 Checklists de Hardening e Boas Práticas de Segurança

Listas de verificação e recomendações para aumentar a segurança de sistemas e redes, cobrindo tópicos como:

*   Configuração segura de sistemas Windows e Linux.
*   Melhores práticas para gerenciamento de senhas e autenticação.
*   Princípios de segurança de rede (firewall, VPN).
*   Conformidade com LGPD e outras regulamentações.

## 🎯 Objetivos

*   Demonstrar proficiência em resolução de problemas de TI e automação.
*   Exibir conhecimento prático em cibersegurança e hardening de sistemas.
*   Fornecer recursos úteis para a comunidade de TI e segurança.
*   Servir como um portfólio dinâmico para futuras oportunidades profissionais.
